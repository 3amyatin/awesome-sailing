# Greece

Greece needs a practical guide focused on Ionian and Aegean trip planning, official references, weather, and reliable local sailing resources.

## Overview
- Use this page as a compact launch point for paperwork, marina selection, and weather checks.
- The links aim at practical cruising decisions rather than general tourism or island-hopping inspiration.

## Official information
- No single official paperwork hub is in this guide yet, so start with the marina and weather references below and verify entry requirements before departure.

## Ports and marinas
- [Greek Marinas](https://greek-marinas.gr/en/) - Public marina directory for Greek cruising destinations.
- Greek Marinas is the main marina reference in this guide.

## Weather
- [Poseidon forecast](https://poseidon.hcmr.gr/) - Official marine forecasting service for Greece.
- [Poseidon mobile app](https://poseidon.hcmr.gr/services/forecast/poseidon-mobile-app) - Mobile access point for the same forecast service.

## Rules and paperwork
- Verify entry, tax, and cruising-tax requirements before departure.
