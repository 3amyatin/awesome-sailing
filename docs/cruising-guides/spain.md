# Spain

Spain is still a lighter guide than Croatia or Italy, but it now has a better practical core: official weather, port-system references, and a Canary Islands starter layer.

## Overview
- Use this page to get from national weather and port references into destination-specific planning.
- The current wave is strongest on official Spain-wide sources plus a first Canary Islands layer.

## Official information
- [Puertos del Estado](https://www.puertos.es/en) - National port-system entrypoint and maritime infrastructure overview. 🇪🇸
- [Puertos Canarios](https://puertoscanarios.es/en/) - Official Canary Islands ports authority. 🇪🇸

## Ports and marinas
- [Puertos del Estado](https://www.puertos.es/en) - Baseline official port authority entrypoint for Spain.
- [Marinas de Espana](https://marinasdeespana.com/) - Spain-wide marina association and useful follow-up for regional marina research. 🇪🇸
- [Puertos Canarios](https://puertoscanarios.es/en/) - Best official starting point for Canary Islands port research. 🇪🇸
- [Puerto de Taliarte](https://puertoscanarios.es/en/project/taliarte-gran-canaria-2/) - Official port page for a Gran Canaria stop. 🇪🇸
- [Puerto de Las Galletas](https://puertoscanarios.es/en/project/las-galletas-2/) - Official port page for a Tenerife-area stop. 🇪🇸
